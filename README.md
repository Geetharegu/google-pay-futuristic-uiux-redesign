##  Project Overview
This project presents a futuristic and modern UI/UX redesign of the Google Pay mobile application. The redesign focuses on improving usability, visual hierarchy, and overall user experience while maintaining simplicity, accessibility, and consistency.

---

##  Objectives
- Enhance visual hierarchy and readability
- Reduce cognitive load during the payment flow
- Create a clean and futuristic user interface
- Improve task completion speed
- Ensure consistent and reusable UI components

---
